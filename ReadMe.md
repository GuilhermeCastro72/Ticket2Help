# Ticket2Help
## Introdução
Este programa foi desenvolvido para um sistema de gestão de tickets de helpdesk para a empresa Ticket2Help. O sistema foi projetado para atender a solicitações de serviços de hardware (HW) e serviços de software (SW), adotando uma arquitetura em 3 camadas (UI, BLL, DAL) e implementado seguindo o padrão de design MVC (Model-View-Controller) utilizando a plataforma WPF (Windows Presentation Foundation) para a interação com o utilizador. O objetivo principal é oferecer uma solução eficiente e intuitiva para a gestão de tickets, facilitando o atendimento e resolução de problemas de hardware e software dentro da empresa.
## Resumo da proposta
A proposta consiste em criar uma plataforma de helpdesk interna que permita aos utilizadores criar e acompanhar tickets de solicitação de serviços de hardware e software, enquanto os técnicos de helpdesk poderão atender a esses tickets e gerar relatórios. A aplicação garante a gestão de acessos através de um sistema de login, assegurando que cada utilizador possua as permissões adequadas às suas funções.
